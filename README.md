# Aulas de Linux

Este repositório contém materiais do curso de **Linux de Ponta a Ponta: Da Filosofia ao Window Manager**, organizados em apresentações Markdown utilizando a extensão Marp para VS Code. As aulas cobrem desde conceitos básicos até tópicos avançados de administração e gerenciamento de sistemas.

## Conteúdo das Aulas

### 01-aula.md
Introdução ao Linux: História, distribuição, instalação e primeiros passos.

### 02-aula.md
Estrutura de diretórios e arquivos no Linux.

### 03-aula.md
Terminal e Gerenciamento de Arquivos: Shell básico, pipes, redirecionamento, comandos essenciais.

### 04-aula.md
Gerenciamento, Administração e Init Systems: Permissões, systemd, sistemas de arquivos, gerenciamento de processos.

### 05-aula.md
Redes e Interfaces Gráficas: Gerenciamento de rede, interfaces gráficas, compositores e window managers.

## Como Usar a Extensão Marp

### Instalação
1. Abra o Visual Studio Code.
2. Vá para a aba de Extensões
3. Pesquise por "Marp for VS Code".
4. Instale a extensão oficial: **Marp for VS Code** (desenvolvida por Marp team).

### Visualizando as Apresentações
1. Abra qualquer arquivo `.md` deste repositório no VS Code.
2. Certifique-se de que o frontmatter do arquivo contenha `marp: true` (já configurado nos arquivos).
3. Clique no ícone do Marp na barra de status inferior do VS Code (ícone de apresentação) ou pressione Ctrl+Shift+P e digite "Marp: Show Preview".
4. A apresentação será aberta em uma nova aba, permitindo navegação entre slides.

### Exportando Apresentações
O Marp permite exportar as apresentações para diversos formatos:
1. Abra o arquivo Markdown no VS Code.
2. Pressione Ctrl+Shift+P e digite "Marp: Export Slide Deck".
3. Escolha o formato desejado: PDF, HTML, PPTX, PNG, etc.
4. Selecione o local de salvamento e confirme.

### Dicas
- As imagens estão organizadas na pasta `imgs/` com subpastas por aula.
- Para editar os slides, modifique o conteúdo Markdown diretamente.
- Use o frontmatter para configurar opções como `paginate: true` para numeração de páginas.
